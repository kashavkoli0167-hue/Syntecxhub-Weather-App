# 🌦 Syntecxhub-Weather-App
A simple and responsive **Weather App** built using **HTML, CSS, and JavaScript**.  
This application fetches **real-time weather data** from the OpenWeather API and displays information such as temperature, weather conditions, humidity, and wind speed.

---

## 🚀 Features

- 🔍 Search weather by city name
- 📍 Get weather using current location
- 🌡 Display current temperature
- 🌡 Show "Feels Like" temperature
- 💧 Humidity information
- 🌬 Wind speed
- 🌤 Dynamic weather icons
- 🎨 Background changes based on weather condition
- 📱 Responsive design for mobile devices

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

---

## ⚙ How It Works

1. The user enters a city name or clicks the **location button**.
2. The app sends a request to the **OpenWeather API**.
3. Weather data is fetched using **fetch() and async/await**.
4. The data is displayed dynamically on the webpage.
